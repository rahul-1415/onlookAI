# Agent Instructions for OnlookAI

## Commits & Attribution

When making commits for this project:

1. **Always ask before committing** — Do not auto-commit
2. **Commit message format:** Imperative summary line, then blank line, then detailed explanation
3. **No trailers** — Do NOT include `Co-Authored-By:` lines or any Claude attribution

## Tech Stack

- Frontend: Next.js 14 + React + Tailwind + shadcn/ui
- Backend: FastAPI (Python) + SQLAlchemy ORM
- Auth: Firebase (frontend) + JWT (backend)
- State: Zustand (frontend), PostgreSQL (backend)
